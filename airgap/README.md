# Air-Gap Packaging & Verification

Offline workflow for the NOC Copilot lab.

## Connected host (build-time egress allowed)

```bash
cd /root/LAB/airgap
bash pull-and-save.sh
```

This pulls any registry images not yet present, then `docker save | xz` every image
(local builds + telemetry stack) into `airgap/images/*.tar.xz` and writes `manifest.txt`.

Move the `airgap/` directory (or just `images/`) to the air-gapped host via USB/SCP/etc.

## Air-gapped host (no internet)

```bash
# Load all images
bash load-offline.sh

# Deploy lab (image-pull-policy: Never already set in clab.yml)
cd /root/LAB/topology
sudo clab deploy -t clab.yml

# Start telemetry stack (all images pinned, no pull attempted)
cd /root/LAB/telemetry
docker compose up -d

# Verify zero runtime egress
bash /root/LAB/airgap/verify-airgap.sh
```

## What verify-airgap.sh checks

1. Every Containerlab node has `image-pull-policy: Never` in `clab.yml`.
2. All required images exist locally (`docker image inspect` passes for each).
3. A 30-second `tcpdump` on `eth0` sees **zero packets** destined for public
   (non-RFC1918) IPs while the lab is running.
4. No background `docker pull` processes are active.
